Esta é uma versão para testes do compilador Plax.
Portanto, ele vai gerar um arquivo .ASM mas não vai apagar.
Este arquivo você poderá analisar como o código final será.
O fato dele não apagar, vai dar erro na próxima compilação.
Então, exclua o arquivo .asm antes de recompilar novamente.

This is a test version of the Plax compiler.
Therefore, it will generate an .ASM file but it will not delete it.
This file you can analyze what the final code will look like.
The fact that it doesn't delete will cause an error in the next compilation.
So, delete the .asm file before recompiling again.
