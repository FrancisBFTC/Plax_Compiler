Here is the following folders:

Analysis - Code to tests and assembly verifications, such as, 64-bit/32-bit comparisions;

Config - Installer to configure the Plax Compiler & association files;

Examples - Plax example codes, for you study some aspects from the language;

Highlight - Files to configure the editors to color the Plax syntax;

Modules - it's called 'Lib Plax' and it's functions to facilitate in the Plax programming;

PlaxLang - This is the language source-code written in C/C++, with the Linker and NASM;

Note: Make sure to follow the folders carefully.
